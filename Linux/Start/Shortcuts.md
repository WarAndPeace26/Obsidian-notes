# Shortcuts
## *Terminal* -> Ctrl+Alt+T
1. Font size
	1. ctrl+shift+*+*
	2. Ctlr+*-*
2. list command
	1. *ls*
3. Clear command
	1. *clear*
	2. ctrl+*L*
4. Kill a process
	1. ctrl+*C*
5. Pause a process
	1. ctrl+z
6. Auto-completion
	1. tab
	2. tab+tab (Recommendations)
7. Close
	1. ctrl+shift+w
